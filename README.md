# JWST M31 (Andromeda) Analysis

Analysis of James Webb Space Telescope observations of the Andromeda Galaxy (M31).

## About this project
- Downloads real JWST observation data from NASA MAST archive
- Analyzes 73 JWST observations of Andromeda
- Visualizes exposure time distribution across different instruments

## Tools
- Python
- astroquery (NASA MAST database)
- pandas
- matplotlib

## Results
JWST has observed M31 with multiple instruments (NIRSpec, NIRCam, MIRI) 
with exposure times ranging from 29 seconds to over 21,000 seconds.
